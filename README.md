# 租车小程序前端

#### 项目介绍
用于租车公司预约的小程序

#### 软件架构

### 后端

- 开发工具：Visual Studio 2015
- 开发语言：C#
- 框架：ABP Zero
- 数据库：MySQL
- NET框架：.net461
- IIS版本：IIS 10

### 小程序端

- 开发工具：微信开发者工具
- UI框架：ZanUI
- 插件：wxparser（富文本显示）
- 其它：阿拉丁统计、fundebug

### 第三方服务

- 腾讯云存储
- 阿里云短信
